# Путеводитель РГАТУ

Цифровой путеводитель для студентов Рыбинского государственного авиационного технического университета имени П.А. Соловьёва.

## О проекте

PWA-приложение (Progressive Web App) с интерактивными поэтажными планами, справочником кафедр и пошаговыми инструкциями по получению документов.

## Технологический стек

- **Фронтенд:** HTML5, CSS3, JavaScript (vanilla)
- **Графика:** SVG-карты этажей (4 файла)
- **Данные:** JSON-структуры (кафедры, документы, контакты)
- **Офлайн:** Service Worker + manifest.json
- **Парсер:** Python (beautifulsoup4, requests)

## Запуск

### Локальный сервер

```bash
cd src
python -m http.server 8080
```

Откройте http://localhost:8080 в браузере.

### Обновление данных

```bash
cd parser
python rsatu_parser.py
```

Данные будут обновлены из официального сайта https://www.rsatu.ru

## Функционал

- **Интерактивная карта** — клик по комнатам, подсветка, карточки с информацией
- **Поиск** — по кабинетам, кафедрам, документам, руководству
- **Справочник институтов** — 3 института, 14 кафедр с контактами
- **Раздел «Документы»** — 12 пошаговых инструкций получения справок и документов
- **Контакты** — телефоны, адреса, email, соцсети
- **Офлайн-режим** — работает без интернета после первой загрузки
- **PWA** — устанавливается на домашний экран
