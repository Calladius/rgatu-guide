# Инструкция для контент-менеджера

## Как обновить данные

Данные хранятся в JSON-файлах в папке `src/data/`. Для обновления не нужен доступ к коду — достаточно отредактировать JSON-файлы.

### Ручное обновление

#### 1. Обновление информации о кафедрах

Файл: `src/data/data_departments.json`

Структура каждого института:
```json
{
  "name": "Название института",
  "shortName": "Аббревиатура",
  "address": "Адрес",
  "phone": "Телефон",
  "email": "Email",
  "director": "ФИО директора",
  "directorTitle": "Должность и степень",
  "floor": 3,
  "room": "327",
  "color": "#1565C0",
  "departments": [
    {
      "name": "Название кафедры",
      "head": "ФИО заведующего",
      "headTitle": "Должность и степень",
      "room": "311",
      "floor": 3,
      "phone": "Телефон"
    }
  ]
}
```

#### 2. Обновление справочника документов

Файл: `src/data/data_documents.json`

Структура документа:
```json
{
  "id": "spravka",
  "name": "Название документа",
  "category": "Категория",
  "description": "Описание",
  "steps": ["Шаг 1", "Шаг 2", "..."],
```

#### 3. Обновление контактов

Файл: `src/data/data_contacts.json`

#### 5. Обновление подразделений

### Обновление SVG-карт

Карты этажей находятся в `src/maps/floor_1.svg` — `floor_4.svg`.

Для добавления/изменения комнаты:
1. Откройте нужный SVG-файл в текстовом редакторе
2. Найдите нужный `<rect>` с атрибутом `data-room`
3. Измените номер комнаты, название или тип

Пример добавления новой комнаты:
```xml
<rect x="100" y="50" width="100" height="200" class="room atif-color" 
      data-room="339" data-type="new-lab" rx="2"/>
<text x="150" y="140" class="room-num">339</text>
<text x="150" y="160" class="dept-label">Новая лаб.</text>
```

### Проверка после обновления

1. Запустите локальный сервер: `python -m http.server 8080`
2. Откройте http://localhost:8080
3. Проверьте все разделы: карта, кафедры, документы, контакты
4. Протестируйте поиск

### Версионирование

После обновления данных измените версию в `src/data/data_meta.json`:
```json
{
  "version": "1.1.0",
  "lastUpdate": "2026-05-13T15:00:00"
}
```

Также обновите `CACHE_NAME` в `src/sw.js`:
```javascript
const CACHE_NAME = 'rgatu-guide-v1.1';
```
